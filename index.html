Entrega N° 1 - JavaScript

Analizador de años

¿En que consiste?

Este proyecto de JavaScript comienza pidiendo al usuario un año y luego ofrece un menu con tres opciones:

  • Opcion 1: Analiza si el año es bisiesto o no.
  • Opcion 2: Calcula cuantos años tendria una persona que nacio en ese año pidiendo como parametros adicionales el dia y el mes de nacimiento.
  • Opcion 3: Indica a que siglo pertence el año.
